# arcgis-cli (arcgis_cli)

**🚧 Work in Progress (WIP) / Early Development 🚧**

This is an early-stage personal/org tool for ArcGIS Online/Enterprise admin tasks.  
Not production-ready yet — APIs, commands, and docs are changing frequently.  
Feel free to watch/star for updates, but **please don't use it in real workflows until v1.0**.

Minimal command-line tool for common ArcGIS Online / ArcGIS Enterprise administration tasks.

Focus: users, groups, basic reporting/exports.

## Project Status

- **Stage**: Early development / Proof-of-concept
- **Working commands**: Basic user search & info (more coming)
- **Not yet implemented**: Group creation, full error handling, tests, packaging polish
- **Expect**: Breaking changes, incomplete docs

## Installation
Requires Python 3.10.x to 3.13.x (to be able to use ArcGIS API for Python 2.4.2)
