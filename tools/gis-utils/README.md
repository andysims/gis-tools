# gis-utils

Simple tool for repetitive GIS operations.

Currently in progress — more utilities coming.

### What's ready (mostly)
- Read CSV with lat/lon or x/y columns
- Generate a new column using incoming x/y and re-projecting

### What will be working on next (no particular order)
- Reproject points to any target EPSG
- Basic extent calculator (coming soon)
- EPSG lookup helper (coming soon)
